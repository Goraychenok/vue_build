# vue-build

## Project setup

### Версия ноды 14


## Запуск для разработки

### Перед запускам установите

```
npm install -g json-server
```

```
npm install -g concurrently
```
### Установите все зависимости
```
npm install
```

### И запустите

```
npm start
```

## Запуск для сборки

### Перед запускам установите

```
npm install -g json-server
```

```
npm install -g concurrently
```

```
npm install -g serve
```
### И запустите

```
npm run start_prod
```
